# Sneh UI Kit

Reusable UI system for all projects.

## Stack
- Next.js (App Router)
- Tailwind CSS v4
- Design Tokens (CSS Variables)
- Component library

## Structure
- `src/design-system/` → tokens, themes, motion CSS files
- `src/components/ui/` → base components like Button
- `src/components/layout/` → layout primitives like Container
- `ai/` → rules and contexts for generative AI assistants

## Next Steps
To use this template in a new project, copy the `design-system` and `components` folders, along with the AI context rules!
